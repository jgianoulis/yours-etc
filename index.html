Data Format

Quick Start

1. Prepare a CSV file with at minimum three columns: letter_num, from, and to. Jen Looper's dataset for Dangerous Liaisons is perfect.
2. Go to Library and click Add a Book. Upload your CSV and fill in the book metadata.
3. Click Visualise on your book card, then select a visualisation.
4. Configure the display using the sidebar controls, then click Download to export.

Required Columns

ColumnTypeDescription
letter_numIntegerDisplay and sort order. Rows are sorted ascending by this value before rendering.
fromStringSender name. Must be consistent — the same character should always appear with the same spelling.
toStringRecipient name. Should match the sender's from spelling exactly.

Optional Columns

ColumnTypeDescription
dateStringFree-text date. Displayed in tooltips only; not parsed or sorted.
topicStringComma-separated keywords. Enables the Topics tag cloud in the Mosaic visualisation.

Name Corrections

If a recipient name in your CSV doesn't match the sender spelling exactly, use Name Corrections in the colour editor modal. Define raw → corrected mappings; they are applied before any visualisation renders and stored with the book record.

Library Export & Import

Use Export Library in the Library toolbar to download all your books as a JSON file. Use Import Library to load it back.

Example CSV

letter_num,from,to,date,topic
1,Cecilia Volanges,Sophia Carnay,"Aug. 3, 17—","friend, dear, write"
2,Marchioness de Merteuil,Viscount de Valmont,"Aug. 4, 17—","time, woman, letter"
